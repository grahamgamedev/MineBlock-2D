MineBlock 2D Readme

Controls

Control Action
Left arrow or A Move left
Right arrow or D Move right
Space Jump
E Crafting
Escape Save and exit
F1 Hide HUD
F2 Screenshot
F3 Show debug info
F4 Save
Delete Delete held items from Hotbar.
Left Click Mine Block/Take from GUI
Right Click Place Block/Put into GUI

Settings

Settings are stored in options.txt and can also be edited from the options menu in the launcher.

Setting Use
Screen width Screen width for the game window in pixels.
Screen hight Screen hight for the game window in pixels.
Sky color In game sky color (R, G, B) the brackets and commas must be there!
Skin filename Skins can be in .bmp, .png or .gif and have to be 90 by 170 pixels and should be in the game folder.
Texture pack folder Texture packs must contain all game textures and be 100 by 100 pixels. They should be in the texture packs folder and this setting is the name of the folder for that individual pack.
